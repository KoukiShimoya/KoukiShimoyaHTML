  1. 影絵ゲーム

    2. 2018年の日本ゲーム大賞に応募しようとして没になった一作。

    影を映すことで、2次元絵に3次元性を持たせることができるのではないのかと思ったのですが、後で2Dで作成すればいいじゃんとなったので没になりました。

    z=0に平らな横長の板を置き、プレイヤーや敵をz=10に置きカメラはz=15の場所に置きます。
    スポットライトをz=8の距離に置き撮影を行っています。
    （zの細かい値は適当で、位置関係把握のために記述しています）

    使用動画

    作成日:2018年3月
  2. Cloaking CAME!

    3. 2018年の日本ゲーム大賞に応募した作品。

    プログラムからモデリング、音楽、効果音までほぼ全てを自分で作った初のゲームです。

  3. ぷちレールシューティング

    4. 株式会社ヒストリア様主催の”ぷちコン”第10回に応募した作品です。

    テーマは”ぷち”、初めてUnreal Engine4を使用してゲームを作りました。

    全くUE4を分からない状態からスタートして、BluePrintを使ってプログラミングをしてゲームを作ったことは良い経験になりました。

  4. シャトルアイ ゼロからの脱出

    5. 2018年11月19日週のUnity1週間ゲームジャムに応募しようとして作成した作品。学内の友人との共同制作です。

    10回目の開催ということもあり、テーマは"10"でした。そこで、1と0の世界の作品を作ろうと考え、AI目線の脱出ゲームを作成しました。

    AIが見る1,0の世界で、宇宙飛行士と協力してSOS信号を送り、脱出を目指します。

    1週間ゲームジャムに応募しようと思いましたが、使用していたcompute shaderがWebGLエキスポート対応していなかったため断念しました。

    こちらからダウンロードして遊ぶことができます。
